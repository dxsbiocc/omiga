# 定时任务管理技能

使用此技能管理 Omiga 的定时任务。

## 命令

### 列出所有任务
```
列出所有定时任务
```

### 查看任务详情
```
查看任务 <task_id> 的详情
```

### 创建任务
```
创建一个定时任务，每天 9:00 发送"早上好"
```

### 删除任务
```
删除任务 <task_id>
```

### 暂停/恢复任务
```
暂停任务 <task_id>
恢复任务 <task_id>
```
