# Excel 处理技能

处理 Excel 文件 (.xlsx, .xls) 的各种操作。

## 功能

- 读取 Excel 数据
- 写入 Excel 文件
- 修改现有 Excel
- 数据筛选和排序
- 公式计算
- 多工作表操作

## 使用方法

```
读取这个 Excel 文件
创建一个新的 Excel 表格
更新 Excel 中的数据
```

## 依赖

- openpyxl: Excel 2010+ (.xlsx)
- pandas: 数据处理
