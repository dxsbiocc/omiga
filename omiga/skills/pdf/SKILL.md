# PDF 处理技能

处理 PDF 文件的各种操作。

## 功能

- 读取 PDF 文本内容
- 提取表格和数据
- 合并多个 PDF
- 分割 PDF
- 创建新 PDF
- 旋转页面
- 添加水印
- OCR 扫描的 PDF

## 使用方法

```
读取这个 PDF 文件
提取 PDF 中的表格
合并这几个 PDF 文件
```

## 依赖

- pypdf: 基本 PDF 操作
- pdfplumber: 文本和表格提取
- reportlab: 创建 PDF
